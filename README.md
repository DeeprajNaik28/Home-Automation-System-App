# Home Automation System App

A working prototype that allows users to control lights and appliances remotely, improving convenience and energy efficiency.

## About the Project

This repository contains only the application-side code for the Home Automation System project.

The project is designed to demonstrate how users can remotely interact with connected devices through a software interface.

## Important Note

Only the app/frontend/backend source code related to the software application is available here.

## Features

- Remote control interface
- Device management system
- User-friendly UI
- Smart appliance interaction
- Prototype-level automation support
